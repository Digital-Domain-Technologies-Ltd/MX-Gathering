# Contribution Guidelines

Thank you for contributing to MX-Gathering!

## Types of Contributions

- Event templates and improvements
- Discussion insights and case studies
- Technical improvements
- Documentation enhancements

## Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit pull request

## Style

- British English (organise, colour, whilst)
- Professional tone
- Clear markdown formatting

See [style-guide-summary.md](style-guide-summary.md) for details.
